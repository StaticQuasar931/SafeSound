# SafeSound

SafeSound is a static, school-safe music web app designed for GitHub Pages. It uses only legal, redistributable audio. This starter version includes original demo tracks generated by the app in the browser so you can test playback, playlists, favorites, focus mode, search, and the license screen without using commercial music or third-party streaming services.

## What This Project Includes

- `index.html` for the app structure
- `style.css` for the responsive UI
- `tracks.js` for the built-in track catalog and metadata
- `script.js` for playback, playlists, favorites, search, filters, and offline registration
- `service-worker.js` for repeat-visit caching

## GitHub Pages Setup

1. Create or open a GitHub repository.
2. Upload all project files to the repository root.
3. Commit and push the files.
4. In GitHub, open `Settings` then `Pages`.
5. Set the source to deploy from your main branch and root folder.
6. Save the setting and wait for GitHub Pages to publish the site.

All paths in this project are relative, so it is GitHub Pages friendly by default.

## Audio Folder Suggestion

If you want to replace the generated demo tracks with real legal audio files, this structure works well:

```text
assets/
  audio/
    calm/
    focus/
    upbeat/
    piano/
    ambient/
    classical/
  covers/
```

Example file paths:

- `assets/audio/calm/gentle-waves.mp3`
- `assets/audio/focus/steady-study.ogg`
- `assets/covers/gentle-waves.jpg`

## How To Replace Demo Tracks With Legal Music

Open `tracks.js` and update each track object:

- Change `src` from a `generated:track-id` value to a real relative file path such as `assets/audio/calm/gentle-waves.mp3`
- Set `cover` to a relative image path if you have cover art
- Keep the legal metadata filled in:
  - `license`
  - `sourceName`
  - `sourceUrl`

Example:

```js
{
  id: "gentle-waves",
  title: "Gentle Waves",
  artist: "Example Artist",
  category: "calm",
  src: "assets/audio/calm/gentle-waves.mp3",
  cover: "assets/covers/gentle-waves.jpg",
  license: "CC0 1.0 Universal",
  sourceName: "Example Archive",
  sourceUrl: "https://example.com/gentle-waves"
}
```

## Important Legal Rule

Only add tracks that are clearly legal to redistribute with your project. Good options include:

- Public-domain music
- CC0 music
- Other royalty-free tracks only when the license clearly allows redistribution in an app or repository

Do not add:

- Commercial copyrighted songs
- Music ripped from Spotify, YouTube, Apple Music, or similar services
- Music with unclear license terms
- User uploads or any upload feature

You should always verify the license terms yourself before adding a new track.

## Notes About The Starter Library

- The included demo tracks are original generated tone studies created by the app for safe testing.
- They are meant as bundled starter content and an example data structure.
- The app shows each track in the built-in About & Licenses section for transparency.

## Accessibility And Device Support

- Keyboard shortcuts:
  - `Space` to play or pause
  - `Left Arrow` to go back 5 seconds
  - `Right Arrow` to go forward 5 seconds
  - `F` to toggle Focus Mode
- Works as a static site without a backend
- Designed to run on school Chromebooks and mobile browsers

## SafeSound Customization Tips

- Add more categories if needed
- Expand the library in `tracks.js`
- Adjust colors and branding in `style.css`
- Keep the app school-safe by avoiding social or unsafe user interaction features
